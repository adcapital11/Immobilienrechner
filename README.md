# Immobilien-Scanner & Rentabilitätsrechner

Ein interaktiver Rentabilitätsrechner für Immobilien in Südtirol, optimiert als Progressive Web App (PWA).

## Features
- Renditeberechnungen (Brutto, Netto, Eigenkapital)
- Unterstützung für Erst-/Zweitwohnsitz-Steuerberechnungen (Registersteuer vs. MwSt.)
- Offline-fähig dank Service Worker
- Datenbank mit Durchschnittsmieten aller 116 Südtiroler Gemeinden

## Starten
### Lokal ausführen (Node.js)
1. Installiere Node.js.
2. Führe die Datei `Start.bat` (Windows) aus oder starte den Server mit:
   ```bash
   node server.js
